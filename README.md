# TASKS
# Быстрый старт

```bash
git clone https://github.com/DmitryKireev69/tasks.git
cd tasks

Установить uv если нету:
curl -LsSf https://astral.sh/uv/install.sh | sh

Если есть выполнить команду
uv sync
После чего активировать виртуальное окружение 

# Для Linux/macOS:
source .venv/bin/activate

# Для Windows CMD:
.venv\Scripts\activate
```
